Personal Picks: Three Best Movies for 2015 so Far

    Mad Max


    Set in a post-apocalyptic world, this movie exhibits no less than the one of the best screenplay, cinematography, and acting in its category. Be advised though that this is not intended for all viewers. Some--in fact most--of the scenes contain gory material. So if you want a more wholesome, family-oriented film, this movie is not for you.
